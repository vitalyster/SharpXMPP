# SharpXMPP

XMPP library for .NET/Xamarin/.NET Core

The library consists of two NuGet packages:

- SharpXMPP: [![NuGet: SharpXMPP](https://img.shields.io/nuget/v/SharpXMPP.svg)](https://www.nuget.org/packages/SharpXMPP/)
